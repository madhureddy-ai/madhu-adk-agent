# Zoo Guide AI Agent 🐾

## 📌 Project Overview
This project is an AI-powered Zoo Guide Agent built using Google ADK and deployed on Google Cloud Run. It helps users explore information about animals such as their habitat, diet, and interesting facts.

---

## 🚀 Features
- Answers user questions about animals
- Uses Wikipedia for general knowledge
- Provides simple and friendly responses
- Deployed on Google Cloud Run

---

## 🛠️ Tech Stack
- Python
- Google ADK (Agent Development Kit)
- LangChain
- Wikipedia API
- Google Cloud Run

---

---

## ⚙️ How it works
1. User enters a question
2. Agent processes the request
3. Fetches information using tools
4. Returns a response to the user

---

## 🎯 Conclusion
This project shows how AI agents can be built and deployed using Google Cloud. It is a simple implementation of an intelligent assistant.
